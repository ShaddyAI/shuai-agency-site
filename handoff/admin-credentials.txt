# Admin Panel Access

## URL
https://yourdomain.com/admin

## Setting Admin Password

The admin panel uses bcrypt-hashed passwords for security.

### Step 1: Generate Password Hash

Run this command locally:
```bash
node -e "console.log(require('bcryptjs').hashSync('YOUR_CHOSEN_PASSWORD', 10))"
```

Replace `YOUR_CHOSEN_PASSWORD` with your actual password.

### Step 2: Set Environment Variable

Add the hash to your Vercel environment variables:

```
ADMIN_PASSWORD_HASH=<hash_from_step_1>
```

### Step 3: Redeploy

Vercel will automatically redeploy when you update environment variables.

## Security Best Practices

1. **Use a Strong Password**
   - Minimum 16 characters
   - Mix of uppercase, lowercase, numbers, symbols
   - Use a password manager

2. **Rotate Regularly**
   - Change password every 90 days
   - Regenerate hash and update env var

3. **Never Commit**
   - Never commit the plain password to git
   - Never commit the hash to git
   - Only store in Vercel environment variables

4. **Limit Access**
   - Only share with authorized team members
   - Use separate passwords for different team members (future enhancement)

## Password Recovery

If you forget the admin password:

1. Generate new hash (Step 1 above)
2. Update ADMIN_PASSWORD_HASH in Vercel
3. Wait for automatic redeploy (~1 minute)
4. Login with new password
